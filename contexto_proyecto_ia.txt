                <p class="lead">El dominio de los <strong>balances de masa</strong> constituye la piedra angular del análisis de procesos en ingenieria petroquímica.                 <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">1. Principios de Conservación</h2>

                <p>La base de todo análisis de procesos químicos descansa en las <strong>leyes de conservación</strong>. Estas leyes establecen que ciertas propiedades permanecen constantes durante una transformación, siempre que se considere un volumen de control apropiado.</p>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">1.1 Ley de Conservación de la Masa</h3>

                <div class="highlight-box-article">
                    <div class="label">Principio Fundamental</div>
                    <p style="font-style: italic; font-size: 1.1rem; margin: 0;">"La materia no se crea ni se destruye, solo se transforma." - Antoine Lavoisier (1789)</p>
                </div>

                <p>En términos matematicos, para cualquier sistema:</p>

                <div class="equation-box">
                    <div class="eq-number">(1.1)</div>
                    $$\text{Masa que entra} - \text{Masa que sale} + \text{Masa generada} - \text{Masa consumida} = \text{Acumulacion de masa}$$
                    <div class="eq-desc">
                        Esta es la <strong>forma general del balance de masa</strong>, valida tanto para sistemas macroscopicos como para el análisis de un punto infinitesimal del fluido.
                    </div>
                </div>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">1.2 Principio de Conservación de la Energia</h3>

                <p>Aunque este artículo se centra en balances de masa, es importante recordar que en procesos petroquímicos reales, los balances de masa y energia estan acoplados:</p>

                <div class="equation-box">
                    <div class="eq-number">(1.2)</div>
                    $$\Delta E = Q - W$$
                    <div class="eq-desc">
                        Donde $Q$ es el calor transferido al sistema y $W$ es el trabajo realizado por el sistema.
                    </div>
                </div>

                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">2. Conceptualizacion de Sistemas</h2>

                <p>Antes de plantear cualquier ecuación de balance, es fundamental definir que estamos estudiando.</p>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">2.1 Definición de Sistema</h3>

                <p>Un <strong>sistema</strong> es una porcion del universo que se selecciona para su análisis, delimitado por una <strong>frontera</strong> (real o imaginaria) que lo separa de los alrededores.</p>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">2.2 Tipos de Sistemas</h3>

                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">3. Formulacion General del Balance de Masa</h2>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">3.1 Balance Macroscópico (Integral)</h3>

                <p>Para un volumen de control fijo en el espacio:</p>

                <div class="equation-box">
                    <div class="eq-number">(3.1)</div>
                    $$\frac{dm_{\text{sistema}}}{dt} = \sum_{j=1}^{N} \dot{m}_{\text{entrada},j} - \sum_{k=1}^{M} \dot{m}_{\text{salida},k} + \dot{m}_{\text{generada}} - \dot{m}_{\text{consumida}}$$
                    <div class="eq-desc">
                        <strong>Variables:</strong><br>
                        $m_{\text{sistema}}$ - Masa total dentro del VC [kg]<br>
                        $\dot{m}_{\text{entrada},j}$ - Flujo masico de entrada por corriente $j$ [kg/s]<br>
                        $\dot{m}_{\text{salida},k}$ - Flujo masico de salida por corriente $k$ [kg/s]<br>
                        $\dot{m}_{\text{generada}}$ - Tasa de generación por reacción [kg/s]<br>
                        $\dot{m}_{\text{consumida}}$ - Tasa de consumo por reacción [kg/s]
                    </div>
                </div>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">3.2 Balance en Términos de Flujos Molares</h3>

                <p>En ingenieria química es frecuente trabajar con moles, especialmente cuando hay reacciones químicas:</p>

                <div class="equation-box">
                    <div class="eq-number">(3.2)</div>
                    $$\frac{dn_{\text{sistema}}}{dt} = \sum_{j=1}^{N} \dot{n}_{\text{entrada},j} - \sum_{k=1}^{M} \dot{n}_{\text{salida},k} + \sum_{i=1}^{R} \dot{\xi}_i \cdot \nu_{\text{componente},i}$$
                    <div class="eq-desc">
                        <strong>Variables:</strong><br>
                        $n_{\text{sistema}}$ - Número total de moles en el sistema [mol]<br>
                        $\dot{n}$ - Flujo molar [mol/s]<br>
                        $\dot{\xi}_i$ - Velocidad de la reacción $i$ [mol/s]<br>
                        $\nu_{\text{componente},i}$ - Coeficiente estequiométrico (positivo si se genera, negativo si se consume)
                    </div>
                </div>

                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">4. Clasificación de los Balances de Masa</h2>

                <p>Los balances de masa se clasifican según dos criterios fundamentales: <strong>el estado temporal del sistema</strong> y <strong>la presencia o ausencia de reacción química</strong>.</p>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">4.1 Según el Estado Temporal</h3>

                <h4 style="font-family: var(--font-serif); font-size: 1.2rem; margin: 1rem 0 0.5rem; color: #2E7D32;">A. Balance en Estado Estacionario (Steady State)</h4>

                <p>Un sistema esta en <strong>estado estacionario</strong> cuando sus propiedades no cambian con el tiempo en ningun punto del volumen de control:</p>

                <div class="equation-box">
                    <div class="eq-number">(4.1)</div>
                    $$\frac{dm_{\text{sistema}}}{dt} = 0 \quad \text{y} \quad \frac{\partial \rho}{\partial t} = 0$$
                </div>

                <div class="highlight-box-article success">
                    <div class="label">Ecuación del Balance Estacionario</div>
                    $$\sum \dot{m}_{\text{entrada}} - \sum \dot{m}_{\text{salida}} + \dot{m}_{\text{generada}} - \dot{m}_{\text{consumida}} = 0$$
                </div>

                <div class="cards-grid-article">
                    <div class="card-article">
                        <div class="card-icon" style="background: #E8F5E9; color: #2E7D32;">&#9989;</div>
                        <h4 style="margin-top: 0;">Caracteristicas</h4>
                        <ul style="margin-left: 20px; color: #616161;">
                            <li>Propiedades constantes en el tiempo</li>
                            <li>Puede haber gradientes espaciales</li>
                            <li>Condicion deseada en plantas petroquimicas</li>
                        </ul>
                    </div>
                    <div class="card-article">
                        <div class="card-icon" style="background: #FFF3E0; color: #E65100;">&#127981;</div>
                        <h4 style="margin-top: 0;">Ejemplo Practico</h4>
                        <p style="color: #616161;">Una columna de destilación operando en condiciones constantes, donde los flujos de destilado y fondos no varian con el tiempo.</p>
                    </div>
                </div>

                <h4 style="font-family: var(--font-serif); font-size: 1.2rem; margin: 1rem 0 0.5rem; color: #E65100;">B. Balance en Estado Transitorio (No Estacionario)</h4>

                <p>Un sistema esta en <strong>estado transitorio</strong> cuando al menos una de sus propiedades cambia con el tiempo:</p>

                <div class="equation-box">
                    <div class="eq-number">(4.2)</div>
                    $$\frac{dm_{\text{sistema}}}{dt} \neq 0$$
                </div>

                <div class="highlight-box-article warning">
                    <div class="label">Ecuación del Balance Transitorio</div>
                    $$\frac{dm_{\text{sistema}}}{dt} = \sum \dot{m}_{\text{entrada}} - \sum \dot{m}_{\text{salida}} + \dot{m}_{\text{generada}} - \dot{m}_{\text{consumida}}$$
                </div>

                <div class="cards-grid-article">
                    <div class="card-article">
                        <div class="card-icon" style="background: #FFEBEE; color: #C62828;">&#9888;&#65039;</div>
                        <h4 style="margin-top: 0;">Caracteristicas</h4>
                        <ul style="margin-left: 20px; color: #616161;">
                            <li>Propiedades dependen del tiempo</li>
                            <li>Requiere resolver EDO o EDP</li>
                            <li>Arranques, paradas, perturbaciones</li>
                        </ul>
                    </div>
                    <div class="card-article">
                        <div class="card-icon" style="background: #E3F2FD; color: #0D47A1;">&#9201;&#65039;</div>
                        <h4 style="margin-top: 0;">Ejemplo Practico</h4>
                        <p style="color: #616161;">El llenado de un tanque de almacenamiento, el arranque de un reactor, o la respuesta ante un cambio en la alimentacion.</p>
                    </div>
                </div>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">4.2 Según la Presencia de Reacción Química</h3>

                <h4 style="font-family: var(--font-serif); font-size: 1.2rem; margin: 1rem 0 0.5rem; color: #0D47A1;">A. Balance Sin Reacción Química (Balance de Masa Físico)</h4>

                <p>Cuando no ocurren transformaciones químicas dentro del volumen de control, la masa de cada componente se conserva individualmente:</p>

                <div class="equation-box">
                    <div class="eq-number">(4.3)</div>
                    $$\dot{m}_{\text{generada}} = \dot{m}_{\text{consumida}} = 0$$
                </div>

                <div class="highlight-box-article">
                    <div class="label">Ecuación General - Sin Reacción</div>
                    $$\frac{dm_{\text{sistema}}}{dt} = \sum \dot{m}_{\text{entrada}} - \sum \dot{m}_{\text{salida}}$$
                </div>

                <p><strong>Para cada componente $i$:</strong></p>

                <div class="equation-box">
                    <div class="eq-number">(4.4)</div>
                    $$\frac{dm_i}{dt} = \sum \dot{m}_{i,\text{entrada}} - \sum \dot{m}_{i,\text{salida}}$$
                </div>

                <div class="process-diagram">
                    <h4 style="margin-top: 0; color: #0D47A1;">Aplicaciónes en Petroquímica</h4>

                <h4 style="font-family: var(--font-serif); font-size: 1.2rem; margin: 1rem 0 0.5rem; color: #2E7D32;">B. Balance Con Reacción Química</h4>

                <p>Cuando ocurren reacciones químicas, la masa total se conserva, pero la masa de cada componente individual puede cambiar.</p>

                <p><strong>Para cada componente $i$ en una reacción:</strong></p>

                <div class="equation-box">
                    <div class="eq-number">(4.5)</div>
                    $$\frac{dm_i}{dt} = \sum \dot{m}_{i,\text{entrada}} - \sum \dot{m}_{i,\text{salida}} + r_i \cdot V$$
                    <div class="eq-desc">
                        $r_i$ - Velocidad de formación del componente $i$ por unidad de volumen [kg/(m&#xb3;s)]<br>
                        $V$ - Volumen del reactor [m&#xb3;]
                    </div>
                </div>

                <p><strong>En términos molares (mas comun en cinética química):</strong></p>

                <div class="equation-box">
                    <div class="eq-number">(4.6)</div>
                    $$\frac{dn_i}{dt} = \sum \dot{n}_{i,\text{entrada}} - \sum \dot{n}_{i,\text{salida}} + R_i \cdot V$$
                    <div class="eq-desc">
                        $R_i$ - Velocidad molar de formación [mol/(m&#xb3;s)]
                    </div>
                </div>

                <p><strong>Relacion con la estequiometria:</strong> Para una reacción general:</p>

                <div class="equation-box">
                    <div class="eq-number">(4.7)</div>
                    $$\nu_A A + \nu_B B \rightarrow \nu_C C + \nu_D D$$
                </div>

                <div class="equation-box">
                    <div class="eq-number">(4.8)</div>
                    $$\frac{\dot{n}_C - \dot{n}_{C,0}}{\nu_C} = \frac{\dot{n}_D - \dot{n}_{D,0}}{\nu_D} = -\frac{\dot{n}_A - \dot{n}_{A,0}}{\nu_A} = -\frac{\dot{n}_B - \dot{n}_{B,0}}{\nu_B} = \dot{\xi}$$
                    <div class="eq-desc">
                        $\dot{\xi}$ - Velocidad de reacción (extensión de reacción por unidad de tiempo)
                    </div>
                </div>

                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">6. Balance Diferencial (Forma Microscópica)</h2>

                <p>Para un análisis detallado de gradientes dentro del equipo, se utiliza la <strong>ecuación de continuidad</strong> en forma diferencial, derivada aplicando el balance a un volumen diferencial $dV = dx \cdot dy \cdot dz$:</p>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">6.1 Forma Vectorial</h3>

                <div class="equation-box">
                    <div class="eq-number">(6.1)</div>
                    $$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \vec{v}) = r$$
                    <div class="eq-desc">
                        $\rho$ - Densidad del fluido [kg/m&#xb3;] &nbsp;|&nbsp; $\vec{v}$ - Vector velocidad [m/s]<br>
                        $r$ - Tasa de generación de masa por unidad de volumen [kg/(m&#xb3;s)] &nbsp;|&nbsp; $\nabla \cdot$ - Divergencia
                    </div>
                </div>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">6.2 Forma en Coordenadas Cartesianas</h3>

                <div class="equation-box">
                    <div class="eq-number">(6.2)</div>
                    $$\frac{\partial \rho}{\partial t} + \frac{\partial (\rho v_x)}{\partial x} + \frac{\partial (\rho v_y)}{\partial y} + \frac{\partial (\rho v_z)}{\partial z} = r$$
                </div>

                <h3 style="font-family: var(--font-serif); font-size: 1.4rem; margin: 1.5rem 0 0.5rem; color: var(--text-primary);">6.3 Casos Particulares</h3>

                <div class="cards-grid-article">
                    <div class="card-article">
                        <div class="card-icon" style="background: #E3F2FD; color: #0D47A1;">&#128167;</div>
                        <h4 style="margin-top: 0;">Fluido Incompresible</h4>
                        <p style="color: #616161; margin-bottom: 12px;">$\rho = \text{constante}$, sin reacción:</p>
                        <div class="equation-box" style="padding: 12px; margin: 0;">
                            $$\nabla \cdot \vec{v} = 0$$
                            $$\frac{\partial v_x}{\partial x} + \frac{\partial v_y}{\partial y} + \frac{\partial v_z}{\partial z} = 0$$
                        </div>
                    </div>
                    <div class="card-article">
                        <div class="card-icon" style="background: #FFF3E0; color: #E65100;">&#9889;</div>
                        <h4 style="margin-top: 0;">Flujo Estacionario</h4>
                        <p style="color: #616161; margin-bottom: 12px;">$\partial \rho / \partial t = 0$:</p>
                        <div class="equation-box" style="padding: 12px; margin: 0;">
                            $$\nabla \cdot (\rho \vec{v}) = r$$
                        </div>
                    </div>
                </div>

                <!-- SECCION 7 -->
                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">7. Estrategia Métodologica para Resolver Problemas</h2>

                <p>Para los estudiantes de Ingeniería Petroquímica, recomiendo el siguiente procedimiento sistemático:</p>

                <div class="steps">
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Definir el Sistema y el Volumen de Control</h4>
                            <p style="margin: 0; color: #616161;">Dibujar el proceso y delimitar claramente la frontera. Identificar todas las corrientes de entrada y salida.</p>
                        </div>
                    </div>
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Recopilar Informacion</h4>
                            <p style="margin: 0; color: #616161;">Flujos conocidos y desconocidos, composiciones de las corrientes, condiciones de operación (T, P), datos cinéticos (si hay reacción).</p>
                        </div>
                    </div>
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Plantear las Ecuaciónes</h4>
                            <p style="margin: 0; color: #616161;">Balance total de masa, balances para cada componente independiente, relaciones adicionales (estequiometria, equilibrio, restricciones).</p>
                        </div>
                    </div>
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Verificar Grados de Libertad</h4>
                            <p style="margin: 0; color: #616161;">Número de incógnitas vs. número de ecuaciones independientes. Debe cumplirse: $N_{\text{ecuaciones}} \geq N_{\text{incógnitas}}$</p>
                        </div>
                    </div>
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Resolver el Sistema</h4>
                            <p style="margin: 0; color: #616161;">Métodos algebraicos (estacionario) o métodos numéricos/diferenciales (transitorio).</p>
                        </div>
                    </div>
                    <div class="step">
                        <div class="step-number"></div>
                        <div class="step-content">
                            <h4 style="margin-top: 0;">Verificar y Validar</h4>
                            <p style="margin: 0; color: #616161;">Comprobar que se cumpla la conservación de la masa total. Analizar si los resultados tienen sentido físico (signos, magnitudes).</p>
                        </div>
                    </div>
                </div>

                <!-- SECCION 8 -->
                <h2 style="font-family: var(--font-serif); font-size: 1.8rem; margin: 2.5rem 0 1rem; color: var(--text-primary);">8. Aplicaciónes Prácticas en la Industria Petroquímica</h2>

                    <div class="label">Reflexión Final</div>
                    <p>El dominio de los balances de masa constituye la <strong>piedra angular</strong> del análisis de procesos en ingenieria petroquímica. La correcta clasificación del sistema -según su estado temporal y la presencia de reacciones químicas- permite seleccionar las herramientas matemáticas apropiadas y simplificar el análisis sin perder rigor.</p>

// ========== SCROLL ANIMATIONS ==========
const observerOptions = {
  threshold: 0.1,
  rootMargin: '0px 0px -50px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
    }
  });
}, observerOptions);

document.querySelectorAll('.fade-in-up').forEach(el => {
  observer.observe(el);
});

// ========== NAVBAR SCROLL EFFECT ==========
const navbar = document.querySelector('.navbar');

window.addEventListener('scroll', () => {
  const currentScroll = window.pageYOffset;

  if (currentScroll > 100) {
    navbar.style.background = 'rgba(255, 255, 255, 0.98)';
    navbar.style.boxShadow = '0 4px 20px rgba(122, 12, 23, 0.08)';
  } else {
    navbar.style.background = 'rgba(255, 255, 255, 0.92)';
    navbar.style.boxShadow = 'none';
  }
});

    e.preventDefault();
    const target = document.querySelector(this.getAttribute('href'));

(function() {
  const materiaButtons = document.querySelectorAll('#materiaFilters .filter-btn');
  const tipoButtons = document.querySelectorAll('#tipoFilters .filter-btn');
  const cards = document.querySelectorAll('.recurso-card');
  const noResults = document.getElementById('noResults');

  if (!materiaButtons.length || !cards.length) return;

  let activeMateria = 'all';
  let activeTipo = 'all';

  function filterCards() {
    let visibleCount = 0;

    cards.forEach(card => {
      const materia = card.dataset.materia;
      const tipo = card.dataset.tipo;

      const matchMateria = activeMateria === 'all' || materia === activeMateria;
      const matchTipo = activeTipo === 'all' || tipo === activeTipo;

      if (matchMateria && matchTipo) {
        card.classList.remove('hidden');
        visibleCount++;
      } else {
        card.classList.add('hidden');
      }
    });

    if (noResults) {
      noResults.style.display = visibleCount === 0 ? 'block' : 'none';
    }
  }

  materiaButtons.forEach(btn => {
    btn.addEventListener('click', () => {
      materiaButtons.forEach(b => b.classList.remove('active'));
      btn.classList.add('active');
      activeMateria = btn.dataset.filter;
      filterCards();
    });
  });

  tipoButtons.forEach(btn => {
    btn.addEventListener('click', () => {
      tipoButtons.forEach(b => b.classList.remove('active'));
      btn.classList.add('active');
      activeTipo = btn.dataset.filter;
      filterCards();
    });
  });
})();
